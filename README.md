# 🧠 Memory Game

An interactive **Memory Matching Game** built with **React + Vite** using **HTML, CSS, and JavaScript**.  
Test your memory skills by flipping cards and matching pairs in this fun and responsive web app! 🎮

---

## 🚀 Live Demo  
🔗 [Play the Game](https://mindhere.netlify.app/)

---

## 🛠️ Tech Stack
- ⚛️ React  
- ⚡ Vite  
- 🧩 HTML, CSS, JavaScript  
- 🎨 Responsive design

---

## ✨ Features
- Flip and match cards to test your memory  
- Tracks score and game progress  
- Responsive layout for all devices  
- Fast and lightweight build using Vite
